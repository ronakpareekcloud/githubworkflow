About Project

This project demonstrates how to deploy a static website on an Amazon EC2 instance using Apache Web Server. The project also includes basic networking and security configuration.

AWS Services Used

  • Amazon EC2
  • Amazon VPC
  • Security Groups
  • Elastic IP
  • Internet Gateway
  • Apache HTTP Server
  • Linux
  • GitHub

Deployment Steps

  1. Create EC2 Instance
  2. Configure Security Group
  3. Connect using SSH
  4. Install Apache
  5. Upload HTML Page
  6. Verify Website
