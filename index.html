AWS EC2 Web Server Project

Hosted by Ronak Pareek

Project Description

This project demonstrates hosting a static HTML website on an AWS EC2 instance using Apache Web Server. It covers instance creation, security configuration, and website deployment.

Services Used

Service Purpose
Amazon EC2 Host Website
Amazon VPC Network
Security Group Firewall Rules
Internet Gateway Internet Access
Apache Web Server

Deployment Steps

  1. Create an EC2 instance.
  2. Allow HTTP and SSH ports.
  3. Connect using SSH.
  4. Install Apache HTTP Server.
  5. Upload the HTML file.
  6. Access the website using Public IP.
